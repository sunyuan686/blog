# [Warp 的窗口管理（分屏）](https://github.com/sunyuan686/blog/issues/21)

[Warp](https://www.warp.dev/)  is the fastest way to build with multiple AI agents—from writing code to shipping it. The best overall coding and terminal agent.

以下快捷键都以Mac 为主，Window 烦请自行Google。
 1.再选中的窗口向右添加窗口 : `Command + D`
